Fortran由于其对数组处理的便利性，以及高效性，被大量使用于大规模科学计算中。为了存储的便利我们往往将数组以`unformatted`的形式写入文件中，本文将介绍如何使用Python读取该文件，得到原数组。

## Fortran输出文件

我们先定义一个三维矩阵`A`，并对`A`赋初值。
```fortran
INTEGER(8) :: A(3,3,3)

do i = 1, 3
  do j = 1, 3
    do k = 1, 3
      A(i,j,k) = i*j*k
    end do
  end do
end do
```

再使`unformatted`输出到文件`array.txt`中。
```fortran
INTEGER(8) :: A(3,3,3)

OPEN(101, file='array.txt', status='unknown', form='unformatted')
write(101) A
WRITE(*,*) "Complete!"
close(101)
```
完整代码如下:
```fortran
PROGRAM put
  INTEGER(8) :: A(3,3,3), i, j, k

  do i = 1, 3
    do j = 1, 3
      do k = 1, 3
        A(i,j,k) = i*j*k
      end do
    end do
  end do

  OPEN(101, file='array.txt', status='unknown', form='unformatted')
  write(101) A
  WRITE(*,*) "Complete!"
  close(101)

END PROGRAM
```

## Python读取文件

因为fortran以`unformatted`形式输出时，文件前四个字节会存储下矩阵的性质，所以需要用`header`
以`unit32`的方式读取一次，这样读取到的数据才是原矩阵。

在fortran代码中，我们定义整数为8个字节，所以在python中，我们要用64位的int来存储数据。最后对A转化为三维数组即可得到原在fortran中的数组。

源代码附上

```python
import numpy as np

nx = 3
ny = 3
nz = 3
A = np.empty((3,3,3),dtype=np.int64)

with open('array.txt','rb') as f:
    header = np.fromfile(f, dtype='uint32', count=1)
    data   = np.fromfile(f, dtype=np.int64, count=nx*ny*nz)

A = np.transpose(np.reshape(data, A.shape))
```
